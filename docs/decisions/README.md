# Decisions

Architectural decisions live in `docs/decisions/` as standalone docs.

## Template (One File Per Decision)

Filename: `ADR-<topic>.md` or `YYYY-MM-DD-<topic>.md`

Suggested structure:
- Context
- Decision
- Alternatives considered
- Consequences
- Rollout/notes (if applicable)
